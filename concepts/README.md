## At a glance

- **IES is a common set of specifications for sharing data across the power sector** — the way UPI is for banking. Data stays in the systems that already hold it; IES specs let any two systems exchange and act on it directly.
- **It works in three steps.** Register (verifiable digital identity) → Discover (interaction protocol) → Exchange (schema, taxonomy and verifiable credentials).
- **It is built on what already exists.** IES uses standards already in use (BIS, CEA, IEC, IEEE) and writes none of its own. The systems you run today keep running — a small "adapter" sits at the edge and ensures conformance.
- **It is live.** The sandbox is running and [four pilot DISCOMs](pilots.md) have built their IES adapter and demonstrated a first set of use cases.
