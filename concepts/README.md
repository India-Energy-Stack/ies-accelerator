## At a glance

- **IES is a common set of specifications for sharing data across the power sector** — the way UPI is for banking. Data stays put; IES specs let any two systems exchange and act on it directly.
- **It works in three steps.** Register (verifiable digital identity) → Discover (interaction protocol) → Exchange (schema, taxonomy and verifiable credentials).
- **It is built on what already exists.** IES uses standards already in use (BIS, CEA, IEC, IEEE) and writes none of its own — your systems keep running, with a small "adapter" at the edge for conformance.
- **It is live.** The sandbox is running and [four pilot DISCOMs](pilots.md) have built their IES adapter and demonstrated a first set of use cases.
